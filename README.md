# Déjà — a coding mentor that remembers you

A CLI coding mentor built on [Cognee](https://github.com/topoteretes/cognee) for the WeMakeDevs "Hangover Part AI: Where's My Context?" hackathon.

Every other AI coding tool wakes up with amnesia. Déjà doesn't.

## Install

```bash
pip install -e .
```

## Commands

```
deja seed      # produce demo graph state
deja start     # cold open — recall who you are
deja chat      # coaching loop with feedback
deja memify    # re-organize the graph
deja forget    # decay + prune
deja ui        # live graph viewer
deja doctor    # verify env
```

Built with Cognee, Python, and Typer.
